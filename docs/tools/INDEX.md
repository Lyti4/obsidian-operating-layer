# Реестр инструментов

**Статус:** `active`

Это полный реестр отслеживаемых `tools/*.py`. Он отвечает на четыре вопроса:
что делает инструмент, где он может писать, чем проверяется и какая инструкция
для него обязательна. Подробный контракт полей находится в
`specs/001-instruction-tree-tool-documentation/contracts/tool-registry.md`.

## Покрытие

- Базовый набор: 58 файлов из Git-индекса ветки `codex/instruction-tree-docs`.
- Документированное покрытие: 58 из 58.
- CLI (командные инструменты): 56.
- Внутренние модули: `tools/_bootstrap.py` и `tools/common.py`.

## Правило изменения

Новый или изменённый инструмент одновременно получает строку в этой таблице,
проверку и ссылку на основное семейное руководство. `approved-write`
(запись в живые данные или состояние хоста) без инструкции с проверяемым
откатом блокирует завершение работы.
